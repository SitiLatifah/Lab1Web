4. Ketik Syntax HTML

``` html
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title></title>
</head>
<body>
```

5. Mengubah Nama Title

``` html
<title>Tag HTML Dasar</title>
```
